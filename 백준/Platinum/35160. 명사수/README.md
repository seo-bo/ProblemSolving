### 문제 설명

<p>동현이는 백발백중하는 명사수이다. 이번에 동현이가 참여하는 대회의 규칙은 다음과 같다.</p>

<ul>
<li>$1$번부터 $N$번까지 번호가 붙은 $N$개의 과녁이 수직선 위에 일렬로 놓여 있다. $i$번 과녁은 초기에 좌표 $i$에 위치하며, <strong>과녁 점수</strong> $S_i$를 가진다.</li>
<li>경기는 총 $Q$라운드로 진행되며, 매 라운드마다 원점에서 화살 한 발을 쏘아 과녁 하나를 맞힌다.</li>
<li>어떤 라운드에 좌표 $j$에 있는 $i$번 과녁을 맞혔다면 해당 라운드의 <strong>라운드 점수</strong>는 다음과 같다.
<ul>
<li><strong>거리 점수</strong>는 미리 정해진 거리 상수 $C$에 대해, $\max\left( C-j,0 \right)$점이다.</li>
<li><strong>라운드 점수</strong>는 <strong>과녁 점수</strong>와 <strong>거리 점수</strong>의 합이다. 즉, $S_i+\max\left( C-j,0 \right)$점이다.</li>
</ul>
</li>
<li>라운드가 끝날 때마다, <strong>라운드 점수</strong>를 계산한 후 모든 과녁은 다음과 같이 이동한다.
<ul>
<li>좌표 $1$에 위치한 과녁은 $N$으로 이동한다.</li>
<li>$1$이 아닌 좌표 $i$에 위치한 과녁은 $i-1$로 이동한다.</li>
</ul>
</li>
<li><strong>최종 점수</strong>는 $Q$번의 <strong>라운드 점수</strong>의 합으로 결정된다.</li>
</ul>

<p>동현이는 백발백중하는 명사수이기 때문에 매 라운드마다 원하는 과녁을 맞힐 수 있다. 동현이가 받을 수 있는 <strong>최종 점수</strong>의 최댓값을 구해보자.</p>

### 입력 

 <p>첫 번째 줄에 과녁의 개수, 라운드 횟수, 거리 상수를 나타내는 정수 $N,Q,C(1\le N,Q,C\le 10^6)$가 공백으로 구분되어 주어진다.</p>

<p>두 번째 줄에 $N$개의 <strong>과녁 점수</strong> $S_1,S_2,\cdots ,S_N(1\leq S_i\leq 10^6)$이 공백으로 구분되어 주어진다.</p>

### 출력 

 <p>첫 번째 줄에 <strong>최종 점수</strong>의 최댓값을 출력한다.</p>
